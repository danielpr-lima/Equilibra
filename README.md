# Diário de Decisões Financeiras (Equilibra)

[cite_start]O **Equilibra** é uma plataforma focada em **saúde financeira comportamental**[cite: 8, 54]. [cite_start]Diferente de aplicativos tradicionais que apenas mostram números, nosso objetivo é promover o autoconhecimento e a reflexão sobre os gatilhos emocionais que levam ao consumo impulsivo[cite: 8, 64].

## 📌 Sobre o Projeto
[cite_start]O projeto atua no combate ao **analfabetismo financeiro comportamental**, que gera ciclos de endividamento, ansiedade e falta de controle sobre o próprio dinheiro[cite: 8, 54]. [cite_start]Através de um diário, o usuário registra não apenas o valor, mas o motivo e o sentimento por trás de cada decisão financeira[cite: 8, 64].

### 🎯 Objetivos de Desenvolvimento Sustentável (ODS)
* [cite_start]**ODS 3 - Saúde e Bem-Estar:** O projeto visa melhorar a saúde mental ao reduzir o estresse e a ansiedade gerados por problemas financeiros[cite: 8, 54].
* [cite_start]**ODS 4 - Educação de Qualidade:** Promove oportunidades de aprendizado e consciência sobre hábitos de consumo ao longo da vida[cite: 9, 11, 65].

## 🛠️ Tecnologias Utilizadas
Para garantir uma interface responsiva e funcional, o projeto utiliza o conjunto fundamental de tecnologias web:

* **HTML5:** Estruturação semântica do conteúdo da plataforma.
* **CSS3:** Estilização, layout e responsividade.
* **JavaScript (JS):** Lógica de programação, manipulação do DOM e interatividade do usuário.
